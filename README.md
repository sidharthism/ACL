========================== Requirements ===========================

BeautifulSoup^4.0.0

============================ Usage ==============================

Add the addresses to address.txt (The program will automatically create one if the file doesn't exist. You can then add the addresses.)

In a shell/cmd/powershell run >python ACL_FortiGuard.py
